*git clone https://github.com/neesara/motivate.git

*You need to have python setup properly and it should be available in path.

*go to the folder motivate
*click on motivate_gui bat file.

To setup so that it will run once the computer bootup do the following :
*************************************************************************
*press windows key + r -> will open 'Run'
*Enter :   shell:startup
*In that folder keep a shortcut to motivate_gui bat file, then it will run at startup.
